🚌 BusBook — Backend

BusBook Backend provides REST APIs for bus search, seat booking, user authentication, and payment processing.

🚀 Features

- 🔐 User Authentication (Login / Signup)
- 🚌 Bus search APIs
- 🪑 Seat availability management
- 🎫 Ticket booking system
- 💳 Payment processing

🛠️ Tech Stack

- ☕ Java
- 🌱 Spring Boot
- 🗄️ MySQL Database
- 🔗 REST API

⚙️ Setup Instructions

1. Clone the repository

git clone https://github.com/samikshapurohit91/busbookBackend.git

2. Open in IDE (Eclipse / IntelliJ)

3. Configure database in application.properties

4. Run Spring Boot application
